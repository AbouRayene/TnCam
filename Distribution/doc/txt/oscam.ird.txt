NAME
       oscam.ird - Irdeto guessing table for oscam

SYNOPSIS
       Irdeto guessing table

DESCRIPTIONS
       <byte3>:<byte4-7>:<CAID>:<SID>
	  Irdeto guessing table by signature

EXAMPLE
	12:0000000a:12ab:cd01
