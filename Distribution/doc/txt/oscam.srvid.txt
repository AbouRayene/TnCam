NAME
       oscam.srvid - service ID configuration file for OSCam

SYNOPSIS
       service ID mappings

DESCRIPTIONS
       CAID[@providerID][,CAID[@providerID]]...:serviceID|[provider]|[name]|[type]|[description]

	  mapping between CAID, provider ID,service ID, provider, name, type and description of service

ANNONTATIONS
       Please use Unix text file format only. Maximum: 10 CAIDs, 5 provider IDs

       You only need the oscam.srvid when using the monitor or the web interface.  For saving memory consumption only
       insert the service IDs you really need. Some external programs use their own oscam.srvid and do not  need  the
       oscam.srvid of OSCam.

EXAMPLES
	0100:1111:provider 1|name 1|TV|TV Name 1
	0100@123456:1111:provider 2|name 2|TV|TV Name 2
	0100@123456,0200:654321:1111:provider 3|name 3|TV|TV Name 3
